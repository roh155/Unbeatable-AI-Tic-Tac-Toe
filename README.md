# 🎮 Neon Tic-Tac-Toe Pro (Unbeatable AI)

A modern, highly interactive, and visually stunning Tic-Tac-Toe game built with **React.js**. It features a neon/cyberpunk UI, a Best-of-3 tournament system, and an **Unbeatable AI Bot** powered by the Minimax Algorithm.

🔗 **Live Demo:** [Play the Game Here](https://rohit-games.pages.dev)  

## ✨ Key Features
* **Unbeatable AI Bot:** Single-player mode features a bot that uses the Minimax algorithm. It calculates all future moves, making it mathematically impossible to beat.
* **Best of 3 Tournament:** It doesn't just stop at one match. The game tracks scores and declares a Series Winner after 3 matches.
* **Two Player Mode (Local):** Play against a friend on the same screen.
* **Neon Cyberpunk UI:** Custom glassmorphism design, neon glow effects, and smooth pop-in animations.
* **Single File Setup:** The entire React app runs flawlessly from a single `.html` file using CDN links and Babel.

## 🛠️ Tech Stack
* **Frontend:** React.js (via CDN), HTML5, CSS3
* **Compiler:** Babel (Standalone)
* **Deployment:** Cloudflare Pages

## 🚀 How to Run Locally (On Your Computer)
Since this project is contained within a single HTML file, running it is incredibly easy:
1. Clone or download this repository.
2. Open the folder in **VS Code**.
3. Install the **Live Server** extension in VS Code.
4. Right-click on the `tic-tac-toe.html` file and select **"Open with Live Server"**.
5. The game will instantly open in your default web browser!

## ☁️ How to Deploy (For Free)
I deployed this project using **Cloudflare Pages**. Here is how you can do it too:
1. Go to [Cloudflare Dashboard](https://dash.cloudflare.com/) and navigate to **Workers & Pages**.
2. Click on **Create application** -> **Pages**.
3. Select **Upload assets** and name your project.
4. Simply drag and drop the folder containing your `.html` file.
5. Click **Deploy Site**. Your game will be live with a global link within seconds!
